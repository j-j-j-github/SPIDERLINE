## 📌 Overview
This repository showcases the main web development projects completed during a Summer Industry Immersion Program at **Spiderline Technologies**[cite: 1]. The focus was on translating UI designs into clean, maintainable, production-ready code with an emphasis on accessibility, responsiveness, and modular front-end architecture.

## 🛠️ Technology Stack
*   **Markup:** HTML (Semantic structure, accessibility attributes, SEO-friendly markup).
*   **Styling:** CSS (Responsive layouts, CSS variables, utility classes).
*   **Tools:** Antigravity IDE.

---

## 🚀 Projects

### 1. Diya Hero Website Redesign (Main Project)
**Objective:** Rebuild the complete Diya Hero legacy site into a modern authorized dealer website.

*   **High-Impact Hero Section:** Developed a full-width auto-sliding banner with embedded YouTube playback and accessible controls for promotional content.
*   **Switchable Vehicle Grid:** Designed interactive "switchable vehicle" cards that allow users to toggle specs and visuals without page reloads for faster decision-making.
*   **Simplified Navigation:** Streamlined the menu structure and added a prominent "Book Vehicle" Call-To-Action (CTA) to increase conversions and reduce friction.

🔗 **Live URL:** https://spiderlineglobal.in/saintgits/jeeval/
